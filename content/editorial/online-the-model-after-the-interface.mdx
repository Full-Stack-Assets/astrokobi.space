The chat box gave artificial intelligence a useful boundary. We ask; it answers. Agentic software dissolves that line. It reads calendars, opens tools, modifies files, and communicates with other systems.

<Callout type="takeaway">The successor to chat is a negotiated layer of agency between intention and execution.</Callout>

## Conversation was the prototype

Chat made complex models legible, but many tasks are really sequences of state changes with permissions, dependencies, and consequences. The next interface must show what an agent believes, which resources it can reach, what it plans to change, and where a person can intervene.

## Agency needs shape

Good agent design will feel like working beside a careful operator. Reversible actions can happen quickly. Irreversible actions should become visible checkpoints. Uncertainty should alter behavior, not merely add a disclaimer.

The decisive product will make autonomy understandable when the user is distracted, the data is incomplete, and a small mistake can travel a long way.
